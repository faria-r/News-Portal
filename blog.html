Important Javascript Questions & Answers

var declarations are globally scoped or function scoped while let and const are block scoped.
var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared.
They are all hoisted to the top of their scope. But while var variables are initialized with undefined, let and const variables are not initialized.

It is unable to use "this" keyword in arrow functions whereas in regular functions it can be used without any disturbance.
If the arrow function has one expression, then the expression is returned implicitly, even without using the return keyword.
But in Regular Function , To Return Something We need to Use Return Keyword.

map() function can fetch each element from the array and do any kind of operation with each element preferred by user and return it .
Filter used to get some specific data from array.
find() can get us only first element from an array which meet with the condition.But filter return all the matched elements with condition.
forEach is Used To loop Through An Array.

A template string, or called template literal, is a practical and useful syntax in JavaScript. It makes our programs neater and better when it comes to string handling.
By using template string we can write multiline string.
We Also can use variable Dynamically By Using Template String.
